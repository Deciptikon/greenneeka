# Greenneeka

Это проект системы управления парком техники на основе локальной радио-сети.

`Внимание! Это не итоговое приложение и не предназначено для промышленного использования, это - прототип!`

## Состав системы управления:

- Монитор (смартфон/ПК).
  Используется для отображения графического интерфейса и удобного управления каждым узлом.

- ESP-gateway (ESP32 + LoRa).
  Используется как master-приёмопередатчик (служит своего рода радиомодемом)

- ESP-device (ESP + LoRa + GPS + otherSensors).
  Используется как slave-приемопередатчики на каждом из устройств, считывают показатели, управляют поведением.

- PWA (кроссплатформенно web-приложение с возможностью локального запуска).
  Представляет собой web-страницу, автоматически подключающуюся к ESP-gateway и предлагающую широкий функционал по мониторингу и конфигурированию всей системы управления.
  Сайт доступен по ссылке [https://deciptikon.github.io/greenneeka/](https://deciptikon.github.io/greenneeka/) (хостинг на github pages)
